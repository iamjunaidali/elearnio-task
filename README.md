### Project Setup

 1. Clone the Project:
 ```sh
git clone git@github.com:iamjunaidali/elearnio-task.git
cd elearnio-task
```
2. Bundle install
    + `bundle install`
6. Run Data Migrations
    + `rails db:setup db:migrate`

System Dependencies:
- Ruby version (3.0.0)
- PostgreSQL (12.4)
